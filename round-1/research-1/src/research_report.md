## Research Findings

This research retrieved exact algorithmic, mathematical, and data specifications across five target domains: (A) HodgeRank [1]: The key system is the sparse least-squares solve lsqr(B1.T, Y) where B1 is the m×n boundary operator (row for edge (i→j): +1 in col j, -1 in col i) and Y_ij = W_ij − W_ji is the net flow. The gradient potential s satisfies L₀s = B₁Y with L₀ = B₁B₁ᵀ (graph Laplacian). The three-component Hodge decomposition is: Y_grad = B₁ᵀs, Y_curl (via B₂ triangular operator), Y_harmonic = remainder — all orthogonal. Curl per triangle: (curl X)(i,j,k) = X_ij + X_jk + X_ki; high curl signals a citation ring. Energy fractions E_grad + E_curl + E_harmonic = 1. (B) CIDRE [2,3]: pip install cidre; API accepts scipy CSR adjacency matrix A[i,j]=weight from i to j, returns Group objects with .donors and .recipients dicts and threshold parameter 0.15. Algorithm uses dcSBM null model λ_ij = (s_i^out·s_j^in·Λ_{g_i,g_j})/(S_{g_i}^out·S_{g_j}^in), Poisson p-values, BH-FDR at α=0.01. Bundled data file edge-table-2013.csv has columns [src, trg, weight] using MAG numeric journal IDs covering ~48K journals. Detects >50% of JCR-suppressed journals. (C) CDFD (arXiv:2506.12546) [4]: decomposition w = c + d with CI = Σc_ij/Σw_ij ∈ [0,1]; BFF algorithm iteratively removes sinks and extracts the maximal divergence-free invariant in polynomial time without cycle enumeration; maximum circularity = minimum cost flow in O((m log n)(m + n log n)). CDFD CI captures both triangular and global-cycle circular flow, unlike HodgeRank curl which is triangle-local only. HHK paper [5] confirms the same approach applies to dense Granger-causality graphs. (D) Clarivate JCR suppression lists [6–11]: 2025 — 20 journals (4 stacking: Applied Organometallic Chemistry/Wiley, Asian Journal of Agriculture & Biology, Chemical Methodologies, Genetic Resources & Crop Evolution/Springer; 16 self-citation); 2024 — 17 journals including stacking pairs Climate Change Economics + Environmental Science & Pollution Research, and Resources Policy + Cuadernos De Economía; 2023 — 4 journals (Genetika, Bioscience Research, Bioinspired-Biomimetic-Nanobiomaterials for stacking; Marketing Theory for self-citation); 2021 — 10 journals (Archivos Latinoamericanos de Nutrition, Journal of Intelligent & Fuzzy Systems, Materials Express for stacking; 7 for self-citation); 2020 — 33 journals (all self-citation); 2018 — 20 journals (6 stacking, 14 self-citation). Total ~100–120 unique suppressed journals across 2018–2025; stacking cartels are the primary Hodge-validation ground truth. (E) OpenAlex API [12,13]: GET /works?filter=primary_location.source.id:SA&select=id,referenced_works&per-page=200&cursor=* for journal works; cursor pagination via meta.next_cursor; free API key at openalex.org/settings/api; for large-scale matrix, use S3 snapshot s3://openalex/data/jsonl/works/ (~330 GB gzip, quarterly free) containing referenced_works and primary_location.source.id per work; OpenAlex CLI (pip install openalex-official) for filtered subsets; CIDRE's bundled 2013 CSV requires no API calls for prototyping.
